# Darren Portfolio

Next.js portfolio website for Darren Hansel Susanto Tan, a Computer Science
student interested in AI/ML and Data Science.

## Pages

- `/` - Home
- `/about` - About
- `/project` - Project
- `/contact` - Contact

## Running The Project

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open `http://localhost:3000`.

## Updating Data

Project and contact data live in `app/lib/portfolio.ts`.

- Update project `href` values if your GitHub repository URLs change.
- Update `darren.fourze@gmail.com` if you want to use a different email.
- Replace the LinkedIn search link with your direct profile URL when available.
